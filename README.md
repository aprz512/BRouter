### 项目如何运行
- 如果你在项目里面使用了插件注册的方式，如下

    ```java
    BRouter.init(this, true);
    ```

    **需要注意：**因为有gradle插件，且工程使用的是本地maven依赖，所以需要先将 `brouter_api` 的插件发布出来，然后才能使用。
    点击 brouter_api -> Tasks -> upload -> uploadArchives 即可将插件发布到项目下的 maven/repository  仓库里面，然后项目即可正常编译。
